# Chrome Bookmarks to Freeplane/Freemind Mindmap

Imports Chrome Bookmarks to Freeplane/Freemind Mindmap <br>
Requires Freeplane/Freemind software to view the files
 
# Steps to Export the Chrome Bookmarks from Chrome:

<ol>
<li>Open Chrome Browser</li>
<li>Enter Ctrl+Shft+O (O for orange) to open the bookmarks manager</li>
<li>Click on the 3 dots icon</li>
<li>Export Bookmarks</li>
<li>Save the HTML file in the same folder as the exe file</li>
</ol>

# Running the Tool:
<ol>
<li>Save the HTML file in the same folder where executable (importChromeBookmarks.exe) is kept</li>
<li>Run the exe </li>
<li>Result file is generated in the same folder.</li>
</ol>
