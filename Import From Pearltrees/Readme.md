# Pearltrees Bookmarks to Freeplane/Freemind Mindmap

Imports Pearltrees Bookmarks to Freeplane/Freemind Mindmap <br>
Requires Freeplane/Freemind software to view the files
 
# Steps to Export the Pearltrees Bookmarks from Pearltrees.com:

<ol>
<li>Go to pearltrees.com and login</li>
<li>Click on hamburger icon on top left corner</li>
<li>Goto More then Settings</li>
<li>Click on Export button save as html</li>
<li>Save the HTML file in the same folder as the exe file</li>
</ol>

# Running the Tool:
<ol>
<li>Save the HTML file in the same folder where executable (importFromPearltrees.exe) is kept</li>
<li>Run the exe </li>
<li>Result file is generated in the same folder.</li>
</ol>
